# Dark Theme Usage Guide - IPatroller System

## Overview
The IPatroller System now features an improved dark theme with enhanced visibility, better contrast, and a modern aesthetic that's easy on the eyes during extended use.

## How to Toggle Dark Theme

### Method 1: System Preference (Automatic)
The application automatically detects your system's theme preference:
- **Windows**: Settings → Personalization → Colors → Choose your color
- **macOS**: System Preferences → General → Appearance
- **Linux**: Depends on your desktop environment

### Method 2: In-App Toggle (If Available)
Look for a theme toggle button in the application header or settings menu (typically a sun/moon icon).

## Accessibility Features

### WCAG Compliance
- **Text Contrast**: All text meets WCAG AA standards (4.5:1 ratio)
- **Large Text**: Headers and large text exceed 3:1 ratio
- **Focus Indicators**: Clear focus states for keyboard navigation
- **Status Communication**: Status conveyed through multiple methods (color, icon, text)

### Color Blind Friendly
- Status is not conveyed by color alone
- Icons accompany all status indicators
- Text labels provide context
- High contrast ensures visibility

### Reduced Eye Strain
- Darker backgrounds reduce blue light exposure
- Softer color palette reduces glare
- Proper spacing prevents visual crowding
- Consistent brightness levels throughout

## Best Practices for Dark Theme Usage

### 1. **Ambient Lighting**
Dark theme works best in:
- Low-light environments
- Evening/night work sessions
- Dimly lit offices
- To reduce screen glare

### 2. **Screen Brightness**
- Adjust screen brightness to match ambient lighting
- Use auto-brightness if available
- Reduce brightness in dark environments

### 3. **Extended Use**
Dark theme is ideal for:
- Long monitoring sessions
- Night shifts
- Reducing eye fatigue
- OLED screens (saves battery)

## Troubleshooting

### Dark Theme Not Activating
1. Check your system theme settings
2. Clear browser cache and reload
3. Check for theme toggle in app settings
4. Ensure JavaScript is enabled

### Colors Look Wrong
1. Ensure you're using a modern browser
2. Update your browser to the latest version
3. Check display calibration settings
4. Try disabling browser extensions

### Text is Hard to Read
1. Adjust screen brightness
2. Increase browser zoom level (Ctrl/Cmd + Plus)
3. Check display scaling settings
4. Consider enabling high contrast mode (if available)

### Elements are Too Dark/Light
1. Check system color profile settings
2. Calibrate your display
3. Adjust monitor contrast settings
4. Report issue to support team
